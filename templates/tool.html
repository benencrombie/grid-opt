SMR Placement Optimization

The interactive tool below can be used to optimize initial "guesses" for SMR placements (see methodology.).

Configure

This tool is currently set to optimize 5 SMR placements, but future revisions may allow for any number to be picked, and for initial "guesses" to be directly modified in the application. If desirable, users can manually change config/grid_config.json with the same json structure to alter the number and initial positions of SMRs. Certain model parameters (e.g., learning rate and step size for gradient descent) were found to be appropriate for this problem to approach convergence while limiting compute. If desirable, these can be modified in config/model_config.json.
